# Repositorio: Sistema de Control y Clasificación

## 1. App Flask - Clasificador con TensorFlow
Ruta: `/ClasificadorFlask`

## 2. App Django - Control de Combustible
Ruta: `/`

- Registro de vehículos (auto y moto)
